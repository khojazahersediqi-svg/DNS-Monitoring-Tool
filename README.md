# DNS-Monitoring-Tool
A Python tool for monitoring DNS servers and detecting connectivity issues.

## Project Objective
This project was developed to improve my understanding of DNS, network monitoring, and cybersecurity concepts by creating a practical DNS monitoring tool using Python.

## Features
- Monitor DNS server availability
- Measure DNS response time
- Detect connection failures
- Log monitoring results
- Simple command-line interface

## Technologies Used
- Python 3
- Socket Programming
- DNS Protocol
- Logging
